# Commit Helper

Quick reference for process-note commits.

## Commit Prefixes
- `log`: daily or session log updates
- `decision`: decision records and updates
- `reading`: paper or concept notes
- `exp-note`: experiment commentary notes
- `report`: deep research report updates
- `meeting`: meeting notes
- `failure`: failure analysis entries
- `idea`: idea capture/refinement
- `docs`: documentation changes
- `chore`: repo maintenance

## Fast Patterns
- `log: add daily-2026-04-24 session recap`
- `decision: document baseline retriever choice`
- `reading: summarize Poison Once, Exploit Forever`
- `exp-note: analyze exp_002_poison_authority prompt drift`

## Optional Commit Body Links
```text
exp_id: exp_002_poison_authority
result_id: res_exp_002_poison_authority_table_v1
research_commit: <short-hash-or-message>
```
