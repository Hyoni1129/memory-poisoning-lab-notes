# How To Use This Repository

## Core Principle
Write process notes close to when the work happens.

## Daily Routine
1. Start a daily note in `03_daily_log/`.
2. Add references to active experiment IDs.
3. Capture decisions in `02_decision_log/` as they are made.
4. Record outcomes and open questions before ending the session.

## Weekly Routine
1. Update `00_research_plan/` with plan changes.
2. Update reading statuses in `08_reading_list/reading-index.md`.
3. Move major insights into experiment notes or deep reports.

## Linking Rule
If a note references a formal result, include:
- `exp_id`
- optional `result_id`
- optional matching commit note from `memory-poisoning-research`
