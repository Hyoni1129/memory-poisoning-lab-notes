# Public Notes Policy

This repository is public. Keep notes transparent but safe.

## Do Not Upload
- credentials, API keys, secrets, or tokens
- confidential meeting details without permission
- restricted datasets or private user data
- private reviewer/advisor comments marked non-public

## LLM-Generated Material Rules
When storing LLM-assisted reports:
- include prompt purpose
- include model/tool used
- include date generated
- include verification status
- include caveats and unresolved claims

## Good Practice
- prefer summaries over raw sensitive transcripts
- redact personal information before commit
- mark uncertain claims clearly as unverified
