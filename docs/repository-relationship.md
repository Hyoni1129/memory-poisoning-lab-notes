# Repository Relationship

This project intentionally uses two separate repositories.

## Roles
- `memory-poisoning-research`: formal, reproducible artifacts
- `memory-poisoning-lab-notes`: process logs, decisions, and research thinking

## Shared Linking Conventions
- shared experiment IDs (`exp_###_<slug>`)
- optional result IDs in notes/commits
- commit body references for lightweight traceability

## Why This Separation Helps
- keeps formal repo clean for publication/portfolio
- keeps process transparent and searchable
- avoids technical coupling while preserving conceptual links
