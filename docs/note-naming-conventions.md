# Note Naming Conventions

Use consistent, searchable filenames.

## Date Format
Always use `YYYY-MM-DD`.

## Standard Patterns
- Daily log: `daily-YYYY-MM-DD.md`
- Decision: `decision-YYYY-MM-DD-topic-slug.md`
- Experiment note: `exp-note-exp_###_slug.md`
- Meeting note: `meeting-YYYY-MM-DD-topic-slug.md`
- Failure note: `failure-YYYY-MM-DD-topic-slug.md`
- Idea note: `idea-YYYY-MM-DD-topic-slug.md`
- Paper note: `paper-<short-paper-key>.md`
- Deep report: `report-YYYY-MM-DD-topic-slug.md`

## Required Header Fields
For most notes include:
- date
- author
- related experiment IDs (if any)
- status (draft / reviewed / verified)
